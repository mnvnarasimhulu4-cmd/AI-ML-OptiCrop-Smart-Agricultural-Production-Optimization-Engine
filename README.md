# OptiCrop: Smart Agricultural Production Optimization Engine

## Project Overview

OptiCrop is an AI/ML-based Smart Agricultural Production Optimization Engine developed to recommend a suitable crop based on soil nutrients and environmental conditions.

The system uses a trained Machine Learning model to analyze agricultural input parameters and predict a suitable crop for the given conditions.

## Input Parameters

The crop recommendation system uses the following parameters:

- Nitrogen (N) level in the soil
- Phosphorus (P) level in the soil
- Potassium (K) level in the soil
- Temperature
- Humidity
- Soil pH value
- Rainfall

## Features

- Machine Learning-based crop recommendation
- Analysis of soil nutrient values
- Analysis of environmental conditions
- Simple and user-friendly web interface
- Input validation
- Fast crop prediction
- Flask-based web application
- Displays the recommended crop based on user inputs

## How the System Works

1. The user enters the required agricultural parameters.
2. The application validates the entered values.
3. The input data is sent to the trained Machine Learning model.
4. The model analyzes the soil and environmental conditions.
5. The system predicts and displays a suitable crop.

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in a code editor.
3. Install the required Python libraries.
4. Open the terminal in the project directory.
5. Run the Flask application:

   python app.py

6. Open the local URL displayed in the terminal.
7. Enter the required soil and environmental parameters.
8. Submit the form to view the crop recommendation.
